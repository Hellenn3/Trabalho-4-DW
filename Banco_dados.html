Biblioteca

Nosso Banco de Dados é uma biblioteca e ele possui 7 tabelas, entre elas as tabelas: Usuários, funcionários, empréstimos, livros, editor, autores, editora. Nessas tabelas estão armazenados os dados da nossa biblioteca. Nosso grupo pensou em fazer uma biblioteca, pois em uma biblioteca temos muitos dados que devem ser armazenados. Para fazer esse banco de dados, pensamos no que uma biblioteca precisava para se organizar, levantamos os dados e fizemos nosso Banco de Dados.
